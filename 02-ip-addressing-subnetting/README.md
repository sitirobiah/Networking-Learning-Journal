🌐 1. What is an IP Address?

An IP address is a unique identifier assigned to each device in a network. It is used to identify and communicate with devices across a network.

📌 Simple example:
192.168.1.10

Without an IP address, devices cannot communicate with each other.

🔷 Types of IP Address
🟢 IPv4

Most commonly used format with 4 number blocks.

Example:

192.168.1.1
32-bit address
Supports around 4 billion addresses
🔵 IPv6

Newer version designed to replace IPv4.

Example:

2001:0db8:85a3::8a2e:0370:7334
128-bit address
Almost unlimited addresses
🧭 2. IP Address Structure (IPv4)

Example:

192.168.1.10

It is divided into 2 parts:

Network part → identifies the network
Host part → identifies the device
🧠 3. What is Subnetting?

Subnetting is the process of dividing a large network into smaller networks.

📌 Why subnetting is important:
Better network organization
Improved security
Efficient IP usage
Easier troubleshooting
📊 4. Subnet Mask

A subnet mask defines which part of the IP address is network and which part is host.

Example:

255.255.255.0

This means:

First 3 parts = Network
Last part = Host
🔢 5. Simple Subnet Example
Network:
192.168.1.0 /24
Devices:
192.168.1.1
192.168.1.2
192.168.1.3

All devices are in the same network and can communicate directly.

🧪 6. Real-World Use Case

Subnetting is used in:

Office networks (separating departments)
School networks (lab separation)
Enterprise systems (security segmentation)
Cloud infrastructure (VPC subnets)
🔄 7. Relation to Networking

IP addressing and subnetting are essential for:

Network configuration
Routing
Security design
Troubleshooting connectivity issues
🎯 Learning Objective

To understand how IP addresses work, how networks are structured, and how subnetting helps organize and optimize network systems.
